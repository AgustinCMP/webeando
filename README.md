 # Webeand🥚 template

**Webeand🥚** is an idea that started not so much time ago, inspired by a very fancy web design I once saw.
This project is the opposite (joke's on me): not meant to be flashy or complex at all. Many mistakes, but still... an idea, simplicity is the correct word.

Here you’ll find a minimal website template, modest but effective way to present something like a résumé (CV), you can use it as CSS-Js playground, or anything you want (garbage can?), the purpose is up to you.

This template it is part of a series of works in persuit of keep practicing HTML, CSS, and JavaScript over time, but I was stuck, so reecently, with some help from AI, I was finally able to move forward and finish <!-- it took me a while, jeez! -->
